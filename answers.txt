Q1: Why might we consider using Git?

Answer: We might consider using Git if we want to collaborate with other developers on a project but we each want to make changes on our own laptops. Additionally, Git lets you track changes made to files over time, which can then allow to revert to those changes easily.

Healthcare professionals could analyze a large amount of patient data to determine when/where outbreaks may occur based on sickness and patient trends.

Q3: What is the difference between Merging and Rebasing?

Answer: The difference between merging and rebasing is that merging creates a new commit with the changes from both branches, but rebasing just moves the changes from one branch to another. Additionally, merging retains the history of the changes, while rebasing rewrites it.
